Projects

  • ncnyt, a hack for hackNY s2011:
      A New York Times reader for the terminal! Written in Python and utilizes libraries like Beautiful Soup and aalib.

  • hmb: Have my babies!, a hack for hackNY f2012: Github repo
      Well, this started off as a joke. Written in Python and Flask, using Temboo API and D3 JS library. This app lets you type in the name of the celebrity and will query Twitter's public tweets for phrases with the celebrity's name and the phrase "have my babies". The data is represented with D3 (one sperm for one tweet).

  • Instagramophone, a hack for PennApps s2013: Github repo
      My friends and I built Instagramophone, which started off as a joke (as most of my hacks seem to) but we had a super fun time with it. It lets you record audio, then apply audio filters. We wrote this in Python and Flask, using the Twilio (TwiML) and Soundcloud APIs. It's hosted on AWS on an Ubuntu server and we used SoX for the audio editing.

  • foursynth, a hack for hackNY s2013.
      My partner and I built a musical synthesizer which produced different sounds based on the number of people checked into trending locations on foursquare. We also implemented a USB keyboard with an Arduino Micro for it. Written in Python, Flask, JS, JQuery, Arduino, using foursquare API and Web Audio API.
